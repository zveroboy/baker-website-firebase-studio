# **App Name**: Baker personal website (Russian Version)

## Core Features:

- Public Homepage: Static welcome content, featured products, contact info, working schedule and delivery zones, and navigation.
- Product Catalog: Grid view of cakes and pastries with filters. Cards link to the Product Detail Page.
- Product Detail Page (with Embedded Order Form): Two-column layout on desktop (Details Left, Form Right), stacked on mobile. Left/Top: Large product image, full description, allergens list, price per unit. Right/Bottom (The Order Form): Configuration: *For Cakes:* Slider/Selector for weight (0.5kg increments). *For Pastries:* Dropdown for quantity (6, 12, 18). *Real-time Price:* Updates instantly as user changes weight/quantity. Customer Info: Name, Phone, Email. Delivery: Pickup (Date/Time) vs. Delivery (Address + Date/Time). Message: Special request text area. Submit: CAPTCHA + "Place Order" button.
- FAQ Section - Public: Public accordion list.
- FAQ Section - Admin: Admin CRUD, reorder.
- Gallery - Public: Public masonry grid.
- Gallery - Admin: Admin upload/reorder.
- Admin Authentication: Login screen (Email/Password).
- Admin Dashboard: Order management (Status, Delivery Fee/Slot inputs), Product CRUD/reorder, Category CRUD/reorder.

## Style Guidelines:

- Warm beige (#F5E7D1).
- Light cream (#FAF9F6).
- Dusty rose (#D3A3A1).
- 'Playfair Display' (Headlines) & 'PT Sans' (Body). Cyrillic Font Support Required.
